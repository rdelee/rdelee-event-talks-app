# Event Talks App

This is a simple single-page web application that displays a schedule of technical talks for a 1-day event.

## How to Run

1.  Clone this repository to your local machine.
2.  Navigate to the directory where you cloned the repository.
3.  Make sure you have Python 3 installed.
4.  Run the following command in your terminal to start a simple web server:

    ```bash
    python3 -m http.server
    ```

5.  Open your web browser and go to `http://localhost:8000` to view the application.
